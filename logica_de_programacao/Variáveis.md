São espaços nomeados na memória do computador usados para armazenar dados que podem ser alterados durante a execução de um programa. Elas permitem que os valores sejam manipulados e reutilizados conforme necessário. Ao declarar uma variável, você está criando um "recipiente"
onde um determinado valor, como números ou textos, pode ser guardado, consultado e modificado ao longo do algoritmo.

### Quais são os tipos de dados que o computador pode armazenar nas variáveis?

Para as linguagens tradicionais como Python, Java, Ruby, JavaScript, C++, C# e tantas outras elas serão definidas como :

- String, caractere, cadeia  - texto 
- Float, real - números flutuantes ( altura, moeda, porcentagem)
- Int - números inteiros (pessoas, animais, prédios)
- booleanas - verdadeiro ou false
Mas para [[Pseudocódigo]] será pouco diferente.
