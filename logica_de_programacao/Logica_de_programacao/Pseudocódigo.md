Pseudocódigo é uma maneira demonstrar algoritmos, mais detalhado do que ocorre no [[Fluxograma]], traz consigo a definição de tipos de variáveis que são utilizadas no algoritmo. 

**Principais Características:**

- **Não executável:** Não é compilado nem rodado pelo computador.
- **Linguagem Simples:** Usa "ler", "escrever", "se", "enquanto".
- **Identação (Recuo):** Importante para organizar a estrutura do código.


### Estrutura básica do pseudocódigo

```
Algoritmo <nome_do_algoritmo>

<declaração_de_variaveis>

Início
	<corpo do algoritmo>
Fim
```
